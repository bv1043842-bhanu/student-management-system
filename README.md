# Java Student Management System

A console-based application built using Java to manage student records efficiently.

## 🚀 Features

* Add new students
* View all student records
* Search by ID and Name
* Update student details
* Delete student records
* Prevent duplicate IDs
* Input validation to avoid errors

## 🛠️ Technologies Used

* Java
* ArrayList
* Basic Data Structures

## 📌 How It Works

This system uses a menu-driven approach where users can perform operations like adding, searching, updating, and deleting student data.

## ▶️ How to Run

1. Compile:
   javac StudentManagementSystem.java
2. Run:
   java StudentManagementSystem
